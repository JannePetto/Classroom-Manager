Class restrictions

Class restrictions and actions

Choose one of your classes, apply a shared preset, set allowed programs and websites for that class, and send class-wide actions.

Admins create and assign classes in the separate class management menu. Teachers only edit restrictions for the classes assigned to their account.

Choosing a preset fills the editor and applies it to the selected class right away. These restrictions are shared for the whole class, so every teacher with access sees the same policy.

Changes in this editor save automatically and are shared with every teacher who can access this class.
